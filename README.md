# Evaluation Symfony

Getting Starded :

 1. git clone project : https://github.com/Elw0od/symfony_eval.git
 2. composer install
 3. php bin/console doctrine:migrations:migrate or using **.sql** file
 4. symfony server:start or php -S localhost:8000 -t public
